2. Use pandas.read_csv and pandas.to_csv

Sometimes, after to_csv, you reload the file with read_csv, it raises exception:

ParserError: Error tokenizing data. C error: Buffer overflow caught - possible malformed input file.

This occurs because there is a text column which contains endline characters. To use this pair of functions properly, explicitly indicate line_terminator by endline character, as follow:

to_csv(lineterminator='\n')
read_csv(line_terminator='\n')
